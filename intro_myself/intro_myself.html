  2. 曾經做過哪些軟體工程技術相關的學習？若有作品請分享給我們。
    3. 我在2022年開始學習python的時候，有試著寫過一些簡單的小遊戲類型的專案，例如與電腦互動的猜數字遊戲。
    這個遊戲會給予玩家5次猜出正確數字的機會，請玩家猜測電腦指定的一個1~100以內的數字。
    程式碼如下：
    print("歡迎來玩猜數字遊戲~\n"
    "答案是1~100隨機的一個整數(5次內猜對即為獲勝)")

    import random
    ans_computer = random.randint(1,100)

    for i in range(1,6):
     print(f"★第{i}次猜測★")
     ans_user = input("請輸入猜測的數字:")

     if i==5 and ans_user.isdigit()==False:
      print("請輸入整數。已超過猜測次數，你輸了~")

     if ans_user.isdigit() == False:
      print("【請輸入整數的數字】\n")
      continue
     # 將使用者的字串輸入轉成int
     ans_user = int(ans_user)

     if i == 5 and ans_user != ans_computer:
      print("\n你輸了～繼續加油吧～")
      print(f"答案是{ans_computer}")
      break

     if ans_user == ans_computer:
      print("恭喜你答對囉～")
      break

     elif ans_user < ans_computer:
      print("提示：再猜大一點\n")
     elif ans_user > ans_computer:
      print("提示：再猜小一點\n")

  5. 請描述一件產生明顯負面情緒的經歷，如何處理該情緒？
    6. 我在大學的時候有參加排球隊，參與過許多比賽。對我來說，在球場上發生失誤，滿容易讓我自己產生不好的心情。 如何處理這個負面情緒的方法是，我總是會告訴自己：「不要被剛剛那一球影響，好好的準備面對下一球，要總是看向未來，不要被過去的失誤影響‧‧‧‧‧‧。」慢慢地心情就比較可以穩住，而心態只要能保持穩定，自身的實力就可以發揮出來。
  6. 關於這份申請網頁，分享一個開發時的技術心得。
    7. 在回答申請網頁的第二題時，我想要把程式碼與專案執行後的成果畫面並排在一起， 我記得去年看過彭彭老師的教學影片，彭彭老師教導使用flex來達成並排的技巧， 可是細節我有些忘記了，於是我重新再看一次老師的教學影片，順利地將排版完成了!
    核心觀念就是，如果想要將「項目」並排，就要在他們的上一層，也就是「容器」，把display的設定設為flex。
  7. 如何看待自身工作和整個社會群體的連結關係?
    8. 這一個題目的範圍相當廣大，也是頗具有哲學性質的議題。
